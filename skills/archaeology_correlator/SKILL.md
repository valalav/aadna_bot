## Назначение

Этот скилл позволяет связывать генетические данные (TMRCA гаплогрупп) с археологическими культурами Кавказа для исторической интерпретации.

## Алгоритм корреляции

### 1. Правила сопоставления

```python
def correlate_culture(tmrca_years):
    """Определяет археологическую культуру по TMRCA"""

    # Молодые ветки (< 1500 лет) - исторический период
    if tmrca_years < 500:
        return {
            "culture": "Кабардинская / Абхазская (формирующийся этнос)",
            "period": "Средневековье - Новое время",
            "confidence": "Высокая",
            "note": "Ветвь соответствует формированию современных этносов"
        }
    elif tmrca_years < 1500:
        return {
            "culture": "Зихийская / Аланская поздняя",
            "period": "Раннее - Позднее средневековье (VI-XIV вв.)",
            "confidence": "Высокая",
            "note": "Формирование адыго-абхазских этносов"
        }

    # Средневековые ветки (1500-3000 лет)
    elif tmrca_years < 2500:
        return {
            "culture": "Аланская / Сарматская",
            "period": "Античность (I в. до н.э. - V в. н.э.)",
            "confidence": "Средняя",
            "note": "Аланы предшествуют осетинам, сарматские связи"
        }
    elif tmrca_years < 3000:
        return {
            "culture": "Скифо-Сарматская",
            "period": "Железный век (VIII-III вв. до н.э.)",
            "confidence": "Средняя",
            "note": "Степной мир, связи с Кавказом"
        }

    # Классические культуры (3000-4500 лет)
    elif tmrca_years < 3800:
        return {
            "culture": "Кобанская поздняя / Меотская",
            "period": "Поздняя бронза (XIII-VIII вв. до н.э.)",
            "confidence": "Средняя",
            "note": "Местные культуры Кавказа"
        }
    elif tmrca_years < 4200:
        return {
            "culture": "Кобанская классическая",
            "period": "Бронзовый век (XV-VIII вв. до н.э.)",
            "confidence": "Высокая",
            "note": "Высокая культура металла, связи с Грецией"
        }
    elif tmrca_years < 4500:
        return {
            "culture": "Кобанская ранняя",
            "period": "Ранняя бронза (XXV-XV вв. до н.э.)",
            "confidence": "Средняя",
            "note": "Формирование кобанской культуры"
        }

    # Энеолит / Ранний бронзовый век (4500-5500 лет)
    elif tmrca_years < 4800:
        return {
            "culture": "Дольменная / Новосвободненская",
            "period": "Энеолит (XXXIII-XXV вв. до н.э.)",
            "confidence": "Средняя",
            "note": "Дольмены как культурный маркер"
        }
    elif tmrca_years < 5200:
        return {
            "culture": "Северно-Кавказская (Ката-Комб)",
            "period": "Энеолит (XXIX-XXV вв. до н.э.)",
            "confidence": "Низкая",
            "note": "Степные миграции в Предкавказье"
        }
    elif tmrca_years < 5500:
        return {
            "culture": "Майкопская поздняя",
            "period": "Ранний бронзовый век (XXXIV-XXXIII вв. до н.э.)",
            "confidence": "Средняя",
            "note": "Упадок майкопской культуры"
        }

    # Майкопская культура (5500+ лет)
    else:
        return {
            "culture": "Майкопская (ранняя/развита)",
            "period": "Энеолит (XXXV-XXX вв. до н.э.)",
            "confidence": "Средняя",
            "note": "Одна из первых комплексных культур Евразии"
        }
```

### 2. Уровни уверенности

| Уровень | Условия | Ограничения |
|---------|---------|-------------|
| **Высокая** | TMRCA < 500 лет или 3500-4200 лет | Хорошая документация культуры |
| **Средняя** | TMRCA в диапазонах 1500-4500 лет | Нескольких культур в периоде |
| **Низкая** | TMRCA > 4500 лет | Древние культуры, мало данных |

### 3. Критерии интерпретации

| TMRCA диапазон | Интерпретация |
|----------------|---------------|
| < 500 лет | Клановые родословные, недавние миграции |
| 500-1500 лет | Формирование современных этносов |
| 1500-3000 лет | Этногенез аланов, предки адыго-абхазов |
| 3000-4500 лет | Кобанская эпоха — местная кавказская культура |
| > 4500 лет | Майкопская культура — ранние миграции |

## Правила использования

### 1. Ограничения корреляции

> [!CAUTION]
> **ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ**
> 
> Корреляция TMRCA с археологическими культурами имеет ограничения:
> 
> 1. **TMRCA ≠ Формирование культуры**
>    - TMRCA — время общего предка протестированных образцов
>    - Не означает, что предок жил именно в этой культуре
> 
> 2. **Миграции и континуитет**
>    - Популяции мигрировали между культурами
>    - Одна культура сменяла другую на той же территории
> 
> 3. **Бутылочное горлышко**
>    - Большой разрыв Formed - TMRCA указывает на долгий период выживания без разветвления
>    - "Туман истории" — может быть несколько культур

### 2. Шаблон формулировки

```markdown
## Исторический контекст

**Возраст ветви:** {TMRCA} лет назад (≈ {Year})

**Предполагаемая культура:** {Culture}

**Интерпретация:**
Возраст ветви ({TMRCA} лет назад) соответствует периоду {Culture} ({period}).
Это не означает, что общий предок жил именно в этой культуре,
но указывает на то, что ветвь формировалась в историческом контексте данного периода.

> [!NOTE]
> Уровень уверенности корреляции: {High/Medium/Low}
> Причина: {reason}
```

### 3. Примеры применения

#### Пример 1: Молодая ветвь (750 лет)

```markdown
**Возраст:** 750 лет назад (≈ 1275 г. н.э.)

**Предполагаемая культура:** Зихийская

**Интерпретация:**
Возраст ветви (750 лет назад) соответствует периоду формирования зихийских племён — предков современных адыго-абхазских народов. Это период средневековья, когда на Западном Кавказе формировалась этническая идентичность предков черкесов и абхазов.

> [!NOTE]
> Уровень уверенности: Высокая
> Причина: Молодая ветвь (< 1000 лет) хорошо коррелирует с историческими источниками
```

#### Пример 2: Средневековая ветвь (2300 лет)

```markdown
**Возраст:** 2300 лет назад (≈ 275 г. до н.э.)

**Предполагаемая культура:** Кобанская поздняя / Меотская

**Интерпретация:**
Возраст ветви (2300 лет назад) соответствует позднему бронзовому веку, периоду кобанской культуры (XIII-VIII вв. до н.э.) и меотской культуры на севере Кавказа. Это период формирования местных кавказских культур с высокими технологиями бронзового литья.

> [!NOTE]
> Уровень уверенности: Средняя
> Причина: В этот период сосуществовали несколько кавказских культур
```

#### Пример 3: Древняя ветвь (4800 лет)

```markdown
**Возраст:** 4800 лет назад (≈ 2775 г. до н.э.)

**Предполагаемая культура:** Майкопская (поздняя) / Новосвободненская

**Интерпретация:**
Возраст ветви (4800 лет назад) соответствует раннему бронзовому веку, периоду упадка майкопской культуры и формирования новосвободненской культуры. Это эпоха дольменов и ранней металлообработки на Кавказе.

> [!NOTE]
> Уровень уверенности: Средняя
> Причина: Трудности с точной датировкой древних культур
```

## Особые случаи

### 1. Mai (G-M201) — Древняя корневая гаплогруппа

```markdown
**Возраст:** 48000 лет назад

**Интерпретация:**
Возраст гаплогруппы G значительно превышает возраст кавказских культур. Присутствие G на Кавказе связано с миграциями верхнепалеолитического периода, задолго до появления майкопской культуры.

> [!CAUTION]
> НЕ коррелировать древние корневые гаплогруппы с конкретными культурами!
```

### 2. G2a1 (M92) — Эффект основателя у осетин

```markdown
**Возраст:** ~4800 лет назад

**Предполагаемая культура:** Майкопская / Раннебронзовая

**Интерпретация:**
Возраст ветви (4800 лет назад) соответствует раннему бронзовому веку, периоду майкопской культуры. Однако современное доминирование G2a1 среди осетинов (~70%) объясняется **эффектом основателя** в средневековье, а не древним происхождением этой гаплогруппы именно у осетин.

> [!NOTE]
> Уровень уверенности: Низкая для этнической интерпретации
> Причина: Эффект основателя искажает связь древнего происхождения с современным распределением
```

### 3. Large Gap (Formed >> TMRCA)

```markdown
**Возраст сформирована:** 14000 лет назад
**Возраст TMRCA:** 2000 лет назад
**Разрыв:** 12000 лет ("Туман истории")

**Интерпретация:**
Ветка формировалась 14000 лет назад (мезолит), но общий предок протестированных образцов жил всего 2000 лет назад (раннее средневековье). Разрыв в 12000 лет указывает на то, что долгий период ветвь выживала без разветвления, возможно, малочисленная популяция или "бутылочное горлышко".

**Предполагаемая культура:** Зихийская (по TMRCA)
**Древнее происхождение:** Мезолит (по Formed)

> [!NOTE]
> Уровень уверенности: Средняя
> Причина: Большой разрыв делает корреляцию по Formed ненадёжной
```

## Ключевые культуры Кавказа

### Майкопская культура (5500-3900 лет назад)

**Характеристики:**
- Ранний бронзовый век
- Курганы с богатым инвентарём
- Торговля с Ближним Востоком
- Металлургия (золото, серебро, медь)

**География:** Западный и Центральный Кавказ

**Связь с гаплогруппами:**
- R1b-Z2103 (степной компонент)
- G2a (местный компонент)
- J2a (ближневосточный компонент)

### Кобанская культура (3000-1300 лет назад)

**Характеристики:**
- Центральный Кавказ (Северная Осетия, Кабарда)
- Бронзовое литьё высокого качества
- Торговля с Грецией и Боспорским царством
- Оружие и орудия труда

**Фазы:**
- Ранняя (3000-2500 лет назад)
- Классическая (2500-1300 лет назад)
- Поздняя (1300-900 лет назад)

**Связь с гаплогруппами:**
- G2a-L1264 (значительное присутствие)
- J2a-M67
- R1a-Z93 (меньше)

### Дольменная культура (3500-2500 лет назад)

**Характеристики:**
- Дольменная архитектура (мегалиты)
- Западный Кавказ, Черноморье
- Загадочное происхождение

**Связь с гаплогруппами:**
- G2a (потенциальная связь)
- J2a (возможный компонент)

### Меотская культура (1200-600 лет назад)

**Характеристики:**
- Нижнее Поволжье, Кубань
- Коневодство
- Связи со степным миром

**Связь с гаплогруппами:**
- R1b-Z2103
- R1a-Z93

### Зихийская культура (500-1200 лет назад)

**Характеристики:**
- Западный Кавказ
- Формирование зихов (предков черкесов)
- Раннесредневековый период

**Связь с гаплогруппами:**
- G2a-L1264 (предки кабардинцев)
- J2a
- R1a, R1b

### Аланская культура (100-500 лет назад)

**Характеристики:**
- Центральный Кавказ
- Предки осетин
- Аланы

**Связь с гаплогруппами:**
- G2a1 (доминирует у осетин)
- J2a
- R1a
