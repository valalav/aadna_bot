В данном списке собраны фамилии из базы данных проекта AADNA, которые относятся к самым молодым и быстрорастущим генетическим ветвям не-кавказского происхождения. ## 🏹 C-Y4541 — 950 лет (Восточно-евразийский / Монгольский след)
Гаплогруппа C имеет явное происхождение из Восточной Евразии (Центральная Азия / Монголия / Сибирь). Ее появление на Кавказе с таким возрастом общего предка (~XI век) идеально коррелирует с эпохой половецких и последующих монгольских миграций.
