## 1. Ландшафт и генетика

### Правило: Проницаемость ландшафта

**Источник:** Gavashelishvili et al., 2021

> Генетическая дифференциация между этническими/субэтническими группами Кавказа коррелирует с **проницаемостью ландшафта** (рельеф, лесной покров, снежный покров), а не с этническими или лингвистическими границами.

### Применение в анализе

| Фактор | Влияние |
|--------|---------|
| Горный рельеф | Снижает генный поток |
| Леса | Увеличивают изоляцию |
| Снежный покров | Зимняя изоляция популяций |

### Примеры для адыгов

- **Зольский район** (казах) — более изолирован → могут быть уникальные гаплогруппы
- **Приэльбрусье** — горный район → меньше смешения
- **Равнинная Кабарда** — больше контактов

## 2. Генетическая преемственность

### Правило: Непрерывность популяций

**Источники:**
- Gavashelishvili et al., 2021
- Margaryan et al., 2017

> Генетическая преемственность в Южном Кавказе прослеживается как минимум **13,000 лет**.

### Северный vs Южный Кавказ

| Регион | Преемственность | Примесь |
|--------|-----------------|---------|
| **Южный Кавказ** | Да (13,000 лет) | Минимальная |
| **Северный Кавказ** | Нарушена | Степная (после бронзового века) |

### Применение

- Адыги (Северный Кавказ) → есть степная примесь
- G2a — местные кавказские гаплогруппы
- R1a, R1b — могут быть от Yamnaya/степные

## 3. Влияние Yamnaya

### Правило: Степные миграции

**Источник:** Gavashelishvili et al., 2021

> Кавказ внёс ~50% генетики в популяцию Yamnaya — степные скотоводы позднего медного — раннего бронзового века.

### Применение

| Гаплогруппа | Происхождение |
|-------------|---------------|
| G2a | Местное кавказское |
| J2 | Местное (древнее) |
| R1a, R1b | Степная примесь |
| L | возможно из Передней Азии |

## 4. Кавказ как барьер

### Правило: Асимметричная проницаемость

**Источник:** Yunusbayev et al., 2012

> Кавказ — **асимметричный полупроницаемый барьер**:
> - Западно-восточные миграции легче
> - Северо-южные миграции труднее

### Направления миграций

```
Запад ←→ Восток  (лёгкие миграции)
   ↑
   ↓ (трудные)
Юг ←→ Север
```

### Применение

- Адыги ближе к западным популяциям
- Связь с Передней Азией через юг
- Меньше связи с северными степями (хотя есть примесь)

## 5. Субэтническая структура

### Правило: Разные профили

**Источник:** Gavashelishvili et al., 2023

> Генетические кластеры коррелируют с языковыми группами.

### Субэтносы адыгов

| Субэтнос | Особенности |
|----------|-------------|
| Кабардинцы | Основной кластер, равнинные |
| Бжедуги | Изолированный район |
| Шапсуги | Причерноморье |
| Тапанта | Горный район |
| Ашхаруа | Абхазская группа |

### Применение

- При анализе учитывать субэтнос
- Бжедуги могут иметь уникальные гаплогруппы
- Тапанта — горные, более изолированные

## 6. Древние культуры и генетика

### Хронология

| Период | Культура | Генетика |
|--------|----------|----------|
| 8-4 вв. до н.э. | Западный Кобан | Местные гаплогруппы |
| 1-3 вв. н.э. | Подкумок-Хамара | Смешение |
| 5-8 вв. н.э. | **Аланская** | Предки адыгов |
| 13-15 вв. | Монгольское нашествие | Степная примесь |

### Применение

- Аланская культура (5-8 вв) — ключевая для адыгов
- TMRCA в диапазоне 1000-3000 лет соответствует аланскому периоду

## 9. Рекомендации для анализа AADNA

### При анализе фамилии учитывать:

1. **Географию** — откуда род
2. **Субэтнос** — бжедуг, кабардинец и т.д.
3. **TMRCA** — соотносить с историческими периодами
4. **Степную примесь** — для R1 гаплогрупп

### Карта субэтносов адыгов

- Кабардинцы: равнина, центр
- Бжедуги: юго-восток, изолированы
- Шапсуги: побережье
- Тапанта: горы
- Ашхаруа: северо-запад
