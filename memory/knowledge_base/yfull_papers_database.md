# YFull Scientific Papers Database

## Источник
База данных научных публикаций по митохондриальной ДНК и гаплогруппам Y-хромосомы.

## Основные категории

### 1. Древняя ДНК (Ancient DNA)
- Палеогеномные исследования различных популяций
- Миграции древних людей
- Генетика доисторических периодов

### 2. Популяционная генетика
- Генетическая структура различных народов
- Миграции и смешение популяций
- Филогеография гаплогрупп

### 3. Митохондриальная ДНК (mtDNA)
- Филогения гаплогрупп мтДНК
- Гаплогруппы: A, B, C, D, E, F, G, H, I, J, K, L, M, N, R, T, U, V, W, X, Y, Z

### 4. Y-хромосома
- Гаплогруппы: A, B, C, D, E, F, G, I, J, L, N, O, Q, R, T

### 5. Медицинская генетика
- Митохондриальные заболевания
- Связь мутаций с болезнями
