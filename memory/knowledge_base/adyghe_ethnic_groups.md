# Адыги (Черкесы) — Этническая структура

## Народ

**Адыги** (самоназвание: адыгэ) = **Черкесы** — единый народ адыгского (абхазо-адыгской) происхождения.

## Основные группы

| Группа | Регион | Статус |
|--------|---------|--------|
| **Адыгейцы** | Адыгея | Региональная группа |
| **Кабардинцы** | КБР | Региональная группа |
| **Черкесы** | Диаспора (Турция, Иордания, Сирия) | Региональная группа |

## Субэтнические группы адыгов

1. **Кабардинцы** — основная группа, КБР
2. **Натухайцы** — западные адыги
3. **Шапсуги** — причерноморские адыги
4. **Абадзехи** — центральные адыги  
5. **Бесленеевцы** — адыги в Адыгее/КБР
6. **Моздокские кабардинцы** — православные кабардинцы

## Происхождение (предки)

- **Зихи** — древнее название
- **Меоты** — античные племена
- **Касоги** — средневековые
- **Каски** — древнее население
- **Синды** — причерноморские племена
- **Керкеты** — горные племена

## Численность

- Россия: ~750,000 (2021)
- Турция: ~3,000,000 (диаспора)
- Всего: ~5,300,000

## Язык

- Кабардино-черкесский язык (адыгэ)
- Адыгейский язык (близкий диалект)

## Для AADNA

При анализе Y-DNA важно различать:
- **Кабардинцы** — основная база проекта
- **Адыгейцы** — ближайшие генетически
- **Черкесы диаспоры** — могут иметь генетические различия из-за изоляции
