## Ключевые выводы

### 1. G2a — доминирующая гаплогруппа
- **G2a — 43-47%** всех трёх групп
- Это **основная гаплогруппа адыгов**

### 2. J2 — вторая по частоте
- J2a* + J2a2* + J2a2a = **15-17%**
- **J2 — ваша линия (J-Y94477) — это местная кавказская гаплогруппа**

### 3. R1a — третья
- R1a1* = 12-14%
- R1b1b2 = до 7%

### 4. J1 — редко
- J1* + J1e* = 3-6%
- В основном у Кабардинцев

### 5. Региональные различия
- **Cherkess** — больше J2a2* (11%)
- **Kabardin** — больше J1 (9%)

## Для J-Y94477

- Вы принадледите к **J2a** (вторая по частоте группа)
- J2a у адыгов: **~15%**
- Это **местная кавказская линия**, не миграция
- Ближайшие родственники — другие J2a адыги/черкесы
