# Резюме: 137 ancient human genomes from across the Eurasian steppes

**Журнал:** Nature (2018)  
**Авторы:** Damgaard P de Barros, Marchi N, Rasmussen S et al.  
**PMID:** 29743675  
**DOI:** 10.1038/s41586-018-0094-2

---

## Основные выводы

### 1. Масштаб исследования
- **137 древних геномов** человека (покрытие ~1x)
- Период: **4,000 лет** — от бронзового века до средневековья
- География: **все Евразийские степи**

### 2. Генетика скифов (Iron Age)
- **Генетически высоко структурированы**
- **Разнообразное происхождение:**
  - Позднебронзовые скотоводы
  - Европейские фермеры
  - Южно-сибирские охотники-собиратели

### 3. Степные миграции
- Скифы **смешались** с восточными степными номадами (Xiongnu)
- Движение на запад в **II-III веке до н.э.**
- Формирование **Гуннских традиций** в IV-V вв. н.э.
- Чума (базальтная к Юстиниановой чуме)

### 4. Средневековые ханства
- Дальнейшее смешение с **Восточноазиатскими группами**
- Превращение степей:
  - **Было:** Индоевропейские языки, Западноевразийское происхождение
  - **Стало:** Тюркские языки, Восточноазиатское происхождение

## Значение для проекта AADNA

### Для понимания генетики адыгов:

1. **Скифы — не единая группа** — разное происхождение разных кланов
2. **Степная примесь пришла с востока** — не с запада
3. **Гунны (IV-V вв)** — ключевое событие для понимания TMRCA
4. **Средневековье** — трансформация генетического ландшафта

### Для датировок:

- J-Y94477 (TMRCA ~900 лет, ~XII век) — **после** гуннских миграций
- Ваша линия — **местная кавказская**, устоявшаяся после степных потрясений
