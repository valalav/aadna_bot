# Резюме: Low-pass DNA sequencing of 1200 Sardinians reconstructs European Y-chromosome phylogeny

**Журнал:** Science (2013)  
**Авторы:** Francalacci P, Morelli L, Angius A et al.  
**PMID:** 23908240  
**DOI:** 10.1126/science.1237947

---

## Основные выводы

### 1. Масштаб исследования
- **1204 сардинских мужчин** секвенировано
- **11,763 SNP** на Y-хромосоме (MSY)
- **6,751 из них** — ранее не наблюдались

### 2. Филогенетическое древо
- Построено **дерево всех основных гаплогрупп Европы**
- Много сардинских специфичных кластеров внутри каждой гаплогруппы

### 3. Датировки
- Калибровка археологическими данными: **~7,700 лет назад** — начальная экспансия Сардинии
- Возраст coalescence: **~180,000 - 200,000 лет назад**
- Согласуется с оценками по мтДНК

### 4. Разные генетические слои
- Возраст узлов выявляет **разные генетические страты** в Сардинии
- Показывает время coalescence с другими человеческими популяциями

---

## Значение для проекта AADNA

### Для понимания генетики адыгов:

1. **Европейская Y-филогения** — важно для понимания происхождения гаплогрупп
2. **Датировки** — 180-200k лет назад — возраст современного человека
3. **Сардиния — референс** — как изолированная популяция показывает "чистые" линии
4. **Калибровка** — метод датировки TMRCA

### Для датировок:

- J-Y94477 (TMRCA ~900 лет) — **очень молодая ветвь** на фоне 200,000 лет
- Sardinia — полезна для калибровки, но не для Кавказа
