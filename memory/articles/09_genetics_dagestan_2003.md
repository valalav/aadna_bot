# Резюме: Genetics and population history of Caucasus populations

**Журнал:** Human Biology, 2003  
**Авторы:** Bulayeva K, Jorde LB, Ostler C, Watkins S, Bulayev O, Harpending H  
**PMID:** 15018034  
**DOI:** 10.1353/hub.2004.0003

---

## Основные выводы

### 1. Исследуемые популяции
- Несколько этнических популяций **Дагестана** (горы Северного Кавказа)
- Методы: **мтДНК** (секвенирование) + **100 полиморфных Alu-инсерций**

### 2. Результаты мтДНК
- мтДНК последовательности **похожи на европейские**
- **Мало обнаруживаемой структуры** в расстояниях между популяциями по мтДНК

### 3. Alu-инсерции
- Популяции Кавказа **больше генетического дрифта**, чем большинство других групп
- Это произошло после расселения современных людей

### 4. Генетические различия
- **Небольшие** генетические различия между популяциями Кавказа
- Сравнимы с различиями между **популяциями Европы**

### 5. Методы анализа
- Сравнение двух методов:
  - Обычные **Fst статистики**
  - Средние частоты **Alu-инсерций**
- Оба метода согласны, если предположить **демографический рост в Африке** до диаспоры

---

## Значение для проекта AADNA

1. **Подтверждает генетическую связь** Дагестана и шире — Северного Кавказа с Европой
2. Показывает **изоляцию и дрифт** популяций в горах — объясняет высокую диверсификацию гаплогрупп
3. Генетические различия между кавказскими популяциями **небольшие** — единый генный пул
4. Контекст для понимания **локальной дифференциации** гаплогрупп у адыгов
