# Резюме: Diverse origin of mitochondrial DNA lineages in Iron Age Black Sea Scythians

**Журнал:** Scientific Reports (2017)  
**Авторы:** Juras A, Krzewinska M, Nikitin AG et al.  
**PMID:** 28266657  
**DOI:** 10.1038/srep43950

---

## Основные выводы

### 1. Происхождение скифов Причерноморья
- **Три источника** мтДНК линий:
  1. Охотники-собиратели
  2. Кочевые популяции Восточной Евразии
  3. Неолитические фермеры Европы

### 2. Гаплогруппы
- **1/3 всех линий** — гаплогруппа **U5** и её субклады
- Общий набор скифов: **H/H5, U5a, A, D/D4, F1/F2**

### 3. Географический градиент
- **Западные линии (U5)** — убывают с запада на восток
- **Восточные линии (A, D)** — убывают с востока на запад

### 4. Связь с предками
- Скифы Причерноморья генетически **схожи с позднебронтовой культурой Срубная** (Срубная-Alasha)
- Это подтверждает археологическую гипотезу: **Срубная → Скифы**

## Значение для проекта AADNA

### Для понимания генетики адыгов:

1. **Скифы — смешанное происхождение** — не единый народ
2. **U5 (1/3)** — важная гаплогруппа для Кавказа
3. **Градиент запад-восток** — объясняет разнообразие на Кавказе
4. **Срубная культура** — вероятные предки скифов Причерноморья

### Для TMRCA:

- Скифы Причерноморья (V-IV вв до н.э.) — **2500 лет назад**
- J-Y94477 (TMRCA ~900 лет) — **после скифов**, в средневековье
